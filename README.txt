HOW TO UPLOAD TO GITHUB PAGES
1. Create a new repository on GitHub, or open your existing repository.
2. Click Add file > Upload files.
3. Open this ZIP file on your computer.
4. Drag ALL files and folders from inside the ZIP into the GitHub upload window.
   Important: upload the contents, not a parent folder wrapped around them.
5. Scroll down and click Commit changes.
6. In the repository, go to Settings > Pages.
7. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
8. Click Save.
9. Wait about 1-3 minutes.
10. Refresh the Pages section until GitHub shows the live website URL.

IF YOU UPDATE THE SITE LATER
- Replace files by choosing Add file > Upload files again.
- If GitHub asks whether to replace files, confirm.
- Click Commit changes.
- Wait a minute and refresh the live site.
